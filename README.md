# Smack Touch Website Review

Static HTML demo for review and sharing.

## Files

- `index.html` - homepage
- `shop.html` - shop landing page
- `collection.html` - collection listing
- `product.html` - product detail page
- `assets/` - images, video, logos, and review media

## Open Locally

For the most reliable local preview, use the included launcher:

- Windows: `START - Windows.bat`
- Mac: `START - Mac.command`

That starts a small local server and opens:

`http://localhost:8765/index.html`

## Share For Review

Recommended option:

1. Push this folder to a GitHub repository.
2. Enable GitHub Pages from the repository settings.
3. Share the GitHub Pages link for visual review.
4. Share the repository link for code/file review.

## GitHub Pages

This project is ready for GitHub Pages because:

- pages are plain `.html`
- assets use local relative paths
- `.nojekyll` is included

After publishing, the main review link will be:

`https://YOUR-USERNAME.github.io/YOUR-REPO/`

## Notes

- If someone downloads the repo as a zip, they should extract it first.
- If local media does not load by double-clicking HTML files, use the launcher scripts instead.
